## Analysis tools

- [`black`](https://pypi.org/project/black/): Linter to apply an auto format for [PEP8](https://www.python.org/dev/peps/pep-0008/).
- [`pylint`](https://pypi.org/project/pylint/): A Python static code analysis tool.
